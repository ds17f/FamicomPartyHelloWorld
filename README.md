## Prerequisits
* FCEUX installed
* Python 3 installed
* CA65 Installed

## Usage
There's a working makefile, so if you just want to run the "game" use:
```
make run
```

You can run the application with some symbols setup for FCEUX's debugger 
by executing
```
make run_debug
```

And you can clean everything up with:
```
make clean
```
